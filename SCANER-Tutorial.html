Getting Started with SCANER

woolingxiang & PoShine

2024-07-27

Tutorial SCANER; Dataset: SmartSeq2 scRNA seq of HNSCC

SCANER represents a novel marker-free approach for the identification of malignant cells within the tumor ecosystem. This method capitalizes on the substantial gene expression variations caused by genomic instability and integrates genes across the entire genome to mitigate the noise introduced by stochastic dropout events at the single-cell resolution. It demonstrates an ability to detect copy number variations (CNVs) in malignant cells with high consistency against whole exome sequencing (WES) results.

Installing the SCANER library

install.packages("devtools")
devtools::install_github('woolingxiang/SCANER')

Sample dataset: Download

SCANER is recommended to be used with a Seurat object containing cell cluster labels as input data.

As example, we will use a dataset containing 5,902 cells from HNSCC patients.

The expression matrix and cell characteristics can be downloaded from the GSE103322.

The expression is provided as log2(TPM/10+1) in the file HNSCC_all_data.txt. Furthermore, the cell-type classification is also available in the file HNSCC_all_data.txt.

Malignancy annotation by SCANER

library(SCANER)
set.seed(12345)
GRCh37.chrpq = SCANER::CpRMAP_GRCh37  ## or SCANER::CpRMAP_GRCh38
scaner = identificationTumorCellsX(seurat_obj=scRNA, cluster.tag='RNA_snn_res.0.4', chrpq=GRCh37.chrpq, GE.frac=0.05, smooth.ceiling=1,
                                   windows=100, cluster.NUM=30, putativeT.frac=0.5, putativeT.cor=0.5, inferCNVBy='putativeTumor2', verbose=T)
